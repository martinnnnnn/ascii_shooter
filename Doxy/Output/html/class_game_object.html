Detailed Description

Recuperation a partir du GameEngine, qui ajoute directement l'object a la scene courante Conteneur de componsants, Ajout de composants au GameObject, qui representent :

  • Soit des fonctions de base (physique, graphique)
  • Soit des scripts permet d'obtenir des comportements pour complexes
